AloeS

The Aloe (Linux) Distribution

AloeS is a purposeless operating system, based on a fork of the Linux kernel rewritten in Malbolge. Its package system allows you to install and remove software if you ask nicely; software is provided in ternary packages or can be built directly from sources with the help of a Malbolge compiler (Not included).

It is available for one platform and one platform only. No, we will not be porting it to anything besides the Nintendo DS. Stop asking.

Please do not follow us anywhere. The dev team are a paranoid bunch who wish to be left alone.

Our build server is updated every 20 years.

Contribute to the AloeS project by making sacrifices before a shrine of the blood goddess. More information can be found in the Handbook.

Not a fork!

It's a spoon!

Just kidding. Unlike trillions of other existing distros, AloeS is not a modification of an existing distribution. We don't actually know who wrote AloeS. After drawing a summoning circle around our intern's laptop and chanting incantations from the Handbook, the source code appeared in a text document labelled "Beware".

Unstable rolling release

Every single update so far has caused major breakages, but we are bound by oaths to the blood goddess to push every package update that appears on Jim's laptop.

Diverse library

There are no limits to the arcane. Any piece of software which could possibly make your life worse is in the AloeS repositories.

Steve

Semi-functional build system

The source code for every package is written in Malbolge. Writing a functional Malbolge compiler in Malbolge is nigh impossible. We have made little progress in creating a build system, but we'd like to see any of you do better. It's good enough.
